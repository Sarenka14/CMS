Konto administratora - admin admin
Można importować tylko pliki znajdujące się w projekcie client/public
W sekcji import/export można importować tylko pliki json w formacie taki jak w pliku json.json
Wszystkie style, artykuły i komentarze są zapisywane w bazie danych.
Style ładują się po przejściu na inny link (style tworzą sie przed odpowiedzią serwera, dlatego na początku są podstawowe)
Wybierz gotowy motyw - wybieranie jednego z trzech gotowych motywów dla głównej sekcji
Stwórz nowy motyw - tworzenie własnego stylu dla głównej sekcji
Zarządzanie menu głownym - tworzenie własnego stylu dla menu, zmiana sposobu wyświetlania menu
Zarządzanie sliderem - ustawienia slidera(czas przejścia, możliwość dodanie zdjęcia i opisu)
Zarządzanie stopką - tworzenie własnego stylu dla stopki
